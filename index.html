DITA Structured Authoring

This course provides a comprehensive introduction to DITA (Darwin Information Typing Architecture) structured authoring.

You will learn the core principles, key components, and applications of DITA to create consistent, reusable, and scalable technical documentation.

Course Audience

This course is designed for junior to mid-level technical writers, content professionals, and documentation decision-makers who are looking to understand and apply structured authoring principles using DITA.
